Dokumentation

Funktionen:

commandLine

Nimmt die Eingabe und duchleuchtet die Eingabe nach validen Befehlen. Wenn es kein gülter Befehl gegeben wird gibt er die Hilfe aus.

getEingabe Return:eingabe

getEingabe erfragt die Eingabe. Wenn die Eingabe leer ist wird die Eingabe erneut erfragt.

innenwinkel Parameter:n Return:alpha

Die innenwinkel Funktion berechnet den innenwinkel.

printHelp Parameter:eingabe

Druckt die Hilfe. Wenn ein Programmname als zweites argument gegeben wird durckt er die Hilfe für diese Programm

vieleck Parameter:eingabe

Zeichnet ein Vieleck. Der n Parameter bestimmt die Anzahl der Ecken. Der r Parameter bestimmt den Umkreisradius. Die Formle um die kantenlänge zuberchenen ist: 2r*sin(𝜋/n)

set Parameter:eingabe

set hat zwei mögliche optionen "set" und "angle". set pos braucht Koardinaten (Reihenfolge: x, y) und set angle braucht einen Winkle.
